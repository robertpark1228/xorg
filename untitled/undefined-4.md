# 유용한 툴

BED 툴 / 서로 겹치는 region 을 자르고 분리하는데 편리함.  
BEDTools, VCFTools, 기타 자르는 툴들의 결과가 조금씩 상이한데 Gene 을 잡는지, Position 을 잡는지에 따라 달라짐. 이는 어쩔 수 없으며 UofM stat 에서도 설명해 놓은 글이 있음.

뭐가 맞다 틀리다 딱히 판단할수는 없으며 개인적인 판단이 필요.
